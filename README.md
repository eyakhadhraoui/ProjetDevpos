# 🚀 Projet Spring Boot - CI/CD avec Jenkins & Docker

## 📌 Description

Ce projet est une application backend développée avec **Spring Boot**, intégrant une chaîne complète de **CI/CD (Intégration et Déploiement Continu)** à l’aide de :

- 🛠️ Jenkins (Pipeline automatisé)
- 🐳 Docker (conteneurisation)
- 📦 Docker Compose (orchestration des services)

🎯 Objectif :  
Automatiser le cycle de vie de l'application, depuis le build jusqu’au déploiement, afin d'assurer une livraison rapide, fiable et reproductible.

---

## 🧰 Technologies utilisées

- Java 17  
- Spring Boot  
- Maven  
- Jenkins  
- Docker  
- Docker Compose  
- Git / GitHub
